Pricing üíµ

PaaS and IaaS are billed based on usage.

Actually understanding your bill is difficult.

Example: Instance is created with 1TB hard drive. You pay for instance time while it is turned on.

Shut it down without deleting it and you still pay for hard drive space!

Saving Money in the Cloud üíµ

Instance reservations*

* only if you know you will be utilizing the instance >80% of the time over 12 months

Saving Money in the Cloud üíµ

Spot/low priority/pre-emptible instances*

* only if you can tolerate losing your instance (checkpoint/restart)

Saving Money in the Cloud üíµ

Data storage tiers and lifecycle policies*

* only if you understand your data access patterns!

Best Practices üåü
